# AirPad

AirPad is an iOS companion app to control a Mac remotely — trackpad, keyboard, app shortcuts, and a live screen view.

## Getting started

- Open `AirPad.xcodeproj` or the workspace in Xcode.
- Build and run on a device or the simulator (some features require device hardware).
